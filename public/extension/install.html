Installation Instructions

1 Download Extension Files

Download the extension package and extract it to a folder on your computer.

Download ScamShield Extension
2 Enable Developer Mode

Open Chrome/Edge and go to chrome://extensions/ or edge://extensions/

Turn on "Developer mode" in the top right corner.

3 Load Extension

Click "Load unpacked" and select the folder where you extracted the extension files.

4 Pin Extension

Click the puzzle piece icon in your browser toolbar and pin ScamShield for easy access.
